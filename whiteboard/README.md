# Whiteboard — 即時白板文件

## 功能

諮詢討論時自動建立 `.local/docs/YYMMDD_主題.md` 白板文件，每次 follow-up 動態更新同一份文件。像開會寫白板一樣，所有資訊集中追蹤。

## 使用方式

- **自動觸發**：當對話為諮詢/建議/規劃性質時自動啟動
- **手動觸發**：`/whiteboard [主題]`

## 白板結構

- 變更/討論摘要（核心資訊表格）
- 已完成（打勾項目）
- 待辦事項（checkbox 追蹤）
- 決策紀錄（日期 + 決策 + 原因）
- 備註（補充資訊）

## 檔案位置

- 白板文件：`.local/docs/whiteboards/YYMMDD_主題描述.md`
- 同主題同日更新既有文件，不另開新檔
