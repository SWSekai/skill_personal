# /md-collab — Markdown 互動式協作

## 功能概述

透過結構化的 Markdown 檔案（核取表 / 填寫表）進行互動式決策協作。Claude 根據主題自動掃描現狀、生成帶有預填建議的決策表，使用者編輯勾選後，Claude 讀取並實作變更。

適用場景：
- 權限配置（單選矩陣）
- 功能開關 / 遷移項目（多選清單）
- 參數配置 / 名稱對照（自由填寫）
- 任何需要多項決策的協作情境

## 使用方式

```
/md-collab <topic> [output_path]
```

| 參數 | 必填 | 說明 |
|------|:----:|------|
| `topic` | Yes | 決策主題，如「API 權限配置」 |
| `output_path` | No | 輸出路徑，預設為專案工作目錄下的適當位置 |

## 執行流程

1. **分析主題** — 掃描相關程式碼，識別決策項目，記錄現狀
2. **生成決策表** — 建立結構化 Markdown，預填建議值
3. **引導編輯** — 告知使用者檔案位置與填寫規則
4. **讀取決策** — 解析勾選結果，生成變更摘要（現狀 → 新值）
5. **確認實作** — 展示摘要、確認後逐項實作

## 互動模式

| 模式 | 適用情境 | 範例 |
|------|---------|------|
| 單選矩陣 | 權限等級、優先順序 | `[ ] low [ ] mid [x] high` |
| 多選清單 | 功能開關、遷移項目 | `[x] 啟用` |
| 自由填寫 | 參數值、名稱對照 | 直接填入文字 |
| 混合模式 | 複雜配置 | 不同區塊用不同模式 |
