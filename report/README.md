# report — 工作報告生成

## 功能概述

從修改紀錄中自動統整生成工作報告，格式針對 2～3 分鐘簡報最佳化。

## 使用方式

```
/report              → 全量報告（所有修改紀錄）
/report weekly       → 近 7 天週報
/report 260311 260319 → 指定期間報告
```

## 報告結構

| 段落 | 用途 | 建議口述時間 |
|------|------|------------|
| 重點成果摘要 | 30 秒掌握全貌 | 30s |
| 功能分類統整 | 依領域分組的表格 | 60s |
| 關鍵技術亮點 | 值得提的設計決策 | 30s |
| 服務影響 | 運維評估 | 15s |
| 待追蹤事項 | 後續行動 | 15s |

## 目錄結構

```
report/
├── README.md    ← 本文件
└── SKILL.md     ← Skill 定義與報告格式規範
```
